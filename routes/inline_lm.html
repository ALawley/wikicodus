HTML Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary. These are used to create new features within texts or images to empasize or link outer elements.

Home

Examples of Inline Elements:

  • <span> element is an inline element that is often used as a container for some text. It can be used to create a of text using style sheets or javascript.
  • <a> tag is used links to internal or external web pages as well as specific areas on a current page.
  • Unlike the previous examples, the <img> image tag embeds a image into your current page whether from an internal or external source.
