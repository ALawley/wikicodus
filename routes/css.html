CSS: cascading style sheets

CSS is a stylesheet language that describes the presentation of an HTML (or XML) document. CSS describes how elements must be rendered on screen, on paper, or in other media. Following is an example of basic CSS syntax:
body {
background-color: #d0e4fe;
}
