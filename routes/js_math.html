JS arithmetic

The JavaScript console can be used to do a lot of math for you...and for me!.

Home

You can open up the JavaScript console in Chrome by pressing CMD-OPT-J. After which you can begin by typing simple equations like 2+2; to start. Afterward we can move on to other functions using operators to divide, like so: 6/2; To find the remainder after division with the "modulo" operator like so: 6%4; multiply: 6*6; And that is just the beginning! Javascript's ability to do math for us creates a lot of opportunity when writing code to make the web dynamic.
