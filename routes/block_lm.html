HTML Block Elements

A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

HTML Block Elements

  • A div is an example of a block element. It is often used to wrap like elements on a page and give those elements consistent styling and position. It is written like so: < div > with a matching closing tag
  • a p tag is written like so: < p > with a closing tag as well and is better known as a paragraph text as it is for creating text elements on the page.
  • headers, h1 through h6 tags are used to create bold text in different sizing that pops out at the top of the page in order create a nice summary of the information below it and welcome the visitor to the page.
  • Form elements are used for interaction between the site and it's creator and the visitor or client to share information, change preferences or needs for the page to accomadate, and to change elements or browser location.
