Cascading Styles

"Cascading" in the CSS context means that because more than one stylesheet rule could apply to a particular piece of HTML, there has to be a known way of determining which specific stylesheet rule applies to which piece of HTML. The rule used is chosen by cascading down from the more general rules to the specific rule required. The most specific rule is chosen.

Cascading Styles

In a specific case if you selected all div's in your file and then added a style to it and then selected a specific div simply by it's class name it may not change that style if it conflicts with your previous styling. Adding a more specific selection such as the name of the parent plus the class or id of the specific div would create priority in the styling cascade to make that rule change the element properly.

non-specific selection:

h2 {
font-size: 32px;
}

specific selection:

.countries h2 {
font-size: 36px;
}
