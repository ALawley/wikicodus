Floats

Floats are used to wrap text around an image, or have a sidebar on one side, or create a page with multiple columns.

Home

floats are used for setting placement of text and images on web pages making it easier for people to read and navigate with a more appealing layout. Stylesheets are used in reference to block elements to present these layouts.

Some code examples are:

  • Creating sidebars
    .sidebar {
    width: 30%;
    float: right;
    background-color: yellow;
    }
  • Dividing pages into columns .column {
    width: 300px;
    float: left;
    }
